# Program 6b - Cookie Management Servlet

## Objective
Create a servlet program to store user name using cookies and display:
- Welcome message
- Number of visits
- List of cookies
- Cookie expiry

## Files
- FirstServlet.java

## Source Code
- Servlet: https://github.com/musabnova/Program6b/blob/master/src/main/java/com/example/FirstServlet.java
## How to Run
1. Run project on Apache Tomcat
2. Open: http://localhost:8080/Program6b/
3. Enter your name and submit
4. Refresh page to see visit count increase
